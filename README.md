# DEXswap Core 

## Clone Repository

`git clone https://github.com/Agin-DropDisco/dexswap-core.git`

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`

## Flatten Contracts

`yarn flattener`

## Flatten Contracts

`yarn flattener`

>> i use flattener for builder in oasis, so they can mixed all the contract without having to imported them one by one .
